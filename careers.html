Level Designer (2D) Core Remote
Layout, pacing, encounter design, and player guidance.
[+]
What you’ll do
  • Block out levels with strong readability and progression beats.
  • Design encounters and resource placement to support the core loop.
  • Collaborate with engineering to implement tools and triggers.
  • Run playtests and iterate based on outcomes.
Nice to have
  • Experience shipping small games or mods; strong taste in pacing.
  • Comfort working inside Unity (tilemaps / prefab placement).
Internal UI/UX Designer Core Remote
Design clear, readable UI and UX flows for small-team games and tools.
[+]
What you’ll do
  • Design HUD, menus, onboarding flows, and in-game UI that prioritize readability and clarity.
  • Create interactive prototypes (Figma/Protopie) and iterate from playtest feedback.
  • Deliver art-ready UI specs and export assets for implementation.
  • Work closely with engineers to implement responsive, performant UI within Unity.
Nice to have
  • Experience with Figma, design systems, and versioned UI libraries.
  • Knowledge of accessibility, localization, and menu economics for small games.
  • Motion/UI polish skills (micro-interactions, transitions) and basic Unity UI experience.
